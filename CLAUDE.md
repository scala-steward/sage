# Working on sage

## Comments

Keep comments minimal. No comments that restate what the code or signature already says. Comment only what is non-obvious: contracts (copy vs zero-copy, mutation rules, poisoning), surprising design decisions, and protocol subtleties. One or two lines, not paragraphs.

## Commits and PRs

Never add AI attribution anywhere: no `Co-Authored-By: Claude` in commit messages, no "Generated with Claude Code" in PR descriptions. Commits and PRs must show only the repository owner.
